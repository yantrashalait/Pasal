Setup Instructions:
1. Django Setup:
   a. create virtualenvironment: virtualenv <virtual_environment_location>/<virtual_environment_name> -p python3
   b. activate virtualenvironment
   c. install the requirements using pip instal -r requirements.txt

2. Database Setup:
   a. Install dependencies:
      sudo apt-get install python3-dev libmysqlclient-dev
   b. Install mysql connector:
      pip install mysqlclient
